Mythos Unleashed

Mythos Unleashed is a Unity-based narrative-driven game focused on systemic gameplay, dynamic world-state interaction, and immersive player feedback.

⸻

Overview

This project explores how player actions can shape the game world in real time through interconnected systems. Instead of static environments, Mythos Unleashed uses reactive systems that influence NPC behavior, environmental feedback, and gameplay flow.

⸻

Core Systems

Global Harmony Index
A dynamic world-state system that tracks player behavior and alters environment, NPC responses, and gameplay feedback in real time.

Procedural Maze Generation
A modular system that generates 3D maze environments using smart prefab placement, room logic, and traversal pathways.

Cognitive State Interface (Diegetic UI)
A UI system where menus are replaced by in-world interactions, keeping players immersed without breaking gameplay flow.

Audio-Reactive Systems
Gameplay elements that respond to sound, light, and player input, creating a feedback loop between player action and environment.
